Frequently asked questions

What is Apache Kafka?

Apache Kafka is a distributed event store and stream-processing platform. It is an open-source system, developed by the Apache Software Foundation and written in Java and Scala.

What is ZooKeeper for?

ZooKeeper is mainly used to track the status of nodes in the Apache Kafka cluster and maintain a list of Kafka topics and messages. In DoubleCloud, we take care of it as part of the managed service.

What costs does using Apache Kafka with DoubleCloud involve?

You only pay for compute and storage, without any additional costs for backups, traffic, or ZooKeeper.

Can I change a cluster configuration after creating it? Will there be any downtime?

Yes, you can change a cluster configuration after it is created with no downtime.

What is BYOC (Bring Your Own Cloud)?

BYOC allows you to deploy Managed Kafka in your AWS or GCP account and control them from your DoubleCloud console. All data, computations, backups, and encryption keys will remain in your cloud account. This is important if you have strict security and compliance requirements or special pricing arrangements with your cloud provider.

By using BYOC, you have the same level of functionality in the DoubleCloud console. We’ll take care of monitoring, backups, patching, configuration, security, and high availability for you.

What’s the difference in pricing between BYOC and managed services in DoubleCloud?

When using BYOC, you pay your cloud provider for the provisioned and consumed resources like virtual machines, disks, storage, and traffic. DoubleCloud automates resource management and takes care of monitoring, backups, patching, configuration, security, high availability, and load balancing provided for under a management fee. To understand how much BYOC would cost, use our calculator and select Bring Your Own Cloud mode.

Learn more about configuring BYOC​

What resources will be provisioned in my account in the BYOC mode?

When you configure BYOC, DoubleCloud first provisions a new VPC, IAM role, and VPC IPv6 CIDR Block in your cloud account. This allows us to create resources in this VPC and with this role.
Later, when adding a new Managed Kafka cluster, DoubleCloud will create the required resources such as virtual machines, disks, encryption keys, and storage buckets.

Does DoubleCloud provide a public API?

We provide the ability to deploy, modify, and manage all four of our services, including creating and deploying charts and dashboards via an API, which is available through the Python SDK, GO SDK or directly through gRPC.
